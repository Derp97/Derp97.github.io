Labs / Raspberry Pi Build

The Observatory

A wall-mounted, local-first Raspberry Pi system designed to provide daily perspective, notes, reflection, astronomy data, motion wake, voice interaction and ambient information.

Project Summary

The Observatory is a physical dashboard appliance built around a Raspberry Pi 5. It is intended to live on the wall, wake when movement is detected, and provide a calm interface for perspective, journaling, notes and daily information.

Core Aim

Not another distracting screen. The goal is to build something useful, grounding and personal: a quiet daily touchpoint that helps me zoom out, capture thoughts and track meaningful progress over time.

Build Progress

The Observatory is being built in staged phases, moving from concept and planning through to hardware assembly and software development.

Phase 01

Planning

Define the purpose, hardware approach, wall-mounted layout, feature set and project direction.

Phase 02

Acquiring

Source the display, storage, camera, sensors, audio hardware, keyboard and mounting components.

Phase 03

Building

Assemble the Raspberry Pi, screen, cooling, storage, PIR sensor, camera, microphone and speakers.

Phase 04

Programming

Build the React interface, FastAPI backend, local database, kiosk mode, hardware services and automation.

Frontend

React, TypeScript and Vite powering a full-screen kiosk interface.

Backend

Python and FastAPI serving local data, notes, hardware events and API responses.

Database

SQLite for local notes, journal entries, future letters and personal history.

Build Philosophy

The Observatory is intentionally local-first. It may pull selected information from external APIs, but it is not designed as a public web service. Private notes, reflections and personal history remain stored locally on the device.

The system should feel more like a calm household fixture than a normal computer: always available, quietly useful, and designed to encourage perspective rather than distraction.
